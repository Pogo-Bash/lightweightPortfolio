## CI pipeline

```
push to main
│
├── commitlint          # commit message linted
├── eslint              # js linting
├── oxlint              # faster rust-based js linting
├── biome               # third linter, contradicts both above
├── prettier            # formatting
├── stylelint           # css linting
├── htmlhint            # html linting
├── markdownlint        # this README is linted
├── ls-lint             # filenames linted
├── editorconfig        # indentation verified
├── cspell              # variable names spellchecked
├── knip                # dead code detection
├── depcheck            # unused dependency detection
├── npm-audit           # known vulnerability check
├── snyk                # second vulnerability check
├── license-checker     # every dep must be MIT or Apache
├── semgrep             # static analysis security scan
├── gitleaks            # scans entire git history for secrets
├── checkov             # terraform + k8s manifest misconfiguration scan
├── kubescape           # NSA/CISA k8s hardening benchmark
├── trivy-fs            # filesystem vulnerability scan
├── zap-scan            # OWASP DAST scan against staging
├── size-limit          # bundle must be ≤50kb or deploy blocked
├── lighthouse-ci       # perf/a11y/seo must be ≥95
├── axe-core            # dedicated accessibility audit
├── type-check          # tsc --noEmit strict mode
└── vitest              # unit tests for the visitor counter
         │
         ▼  all 27 must pass
    docker build (multi-stage, final image: nginx:alpine)
         ▼
    trivy image scan
         ▼
    push → ghcr.io/swap/portfolio:${{ github.sha }}
         ▼
    helm values.yaml updated (image tag bumped via git commit)
         ▼
    argocd detects drift, syncs cluster
         ▼
    rolling deploy across 3 replicas (maxUnavailable: 0, maxSurge: 1)
         ▼
    metallb → haproxy → ingress-nginx → istio mtls → pods
         ▼
    prometheus scrapes new pods
         ▼
    grafana goes green
         ▼
    site says hi
```
